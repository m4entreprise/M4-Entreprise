# M4 Entreprise – Design Agency++

## Objectif global

Créer un site web moderne, premium, vendeur, original, avec un haut niveau de design et d’expérience utilisateur. Niveau agence internationale.

## Stack technique

* **Vue 3 (Composition API)**
* **Vite**
* **Tailwind CSS + tokens personnalisés**
* **Headless UI** pour les composants structurels
* **Motion One ou GSAP** pour les animations premium
* **Phosphor Icons**
* **Three.js** (léger, uniquement où utile)
* **Polices** : Satoshi / Inter + Clash Display

---

## Design System (Premium++)

### Typographies

* Titres : **Clash Display** / Monument Extended
* Paragraphes : **Satoshi** / Inter

Échelle :

* H1 : 64–96 px
* H2 : 40–56 px
* H3 : 28–32 px
* Body : 18–20 px

### Couleurs

* Fond : `#0D0D0D`
* Primaire : `#6C4CFF`
* Accent : `#E2DEFF`
* Contraste : `#FFFFFF`
* États : Vert `#38E6A3` / Rouge `#F2545B`

### Style général

* Ultra minimaliste
* Glassmorphism léger
* Dégradés directionnels subtils
* Spacing large et structuré

## Animations (niveau agence++)

* Micro-interactions au hover

* Header dynamique (blur + shrink)
* Transition page → page en fondu croisé
